What is actually here?

A GitHub-backed snapshot of the public namespace: five original projects, six selected forks, seven Python-led repos, and one plain HTML Pages site.

  • Five source projects live here.

    The original set currently includes this site plus Hermes plugins for Mycelium provenance, Memory Wiki, memory governance, and delegated Claude Code runtime workflows.

  • Six forks track adjacent systems.

    The fork set covers Hermes WebUI, Open Design, Hermes War Room, DeepTutor, hermes-gizmo, and gbrain: adjacent upstreams kept close for inspection and adaptation.

  • One static front door.

    hermegeddon.github.io is a plain HTML GitHub Pages repo with a custom domain, so the public site remains easy to diff, preview, and improve.

Source first. Tools second.

The page starts with public repository data, then points to the few project resources that help someone verify or improve it.

  • Design files
    The page is the product.

    index.html is the canonical public page; CNAME binds hermegeddon.dev; README.md records local preview with Python's HTTP server.

  • Skills
    Use specialist review only when it helps.

    Creative direction and design review are the useful lane here; media, deck, and ad skills wait until the page needs those surfaces.

  • Templates
    Export paths stay optional.

    Screenshot, accessibility, PDF, or hosting templates are useful for review and publishing, not for inventing new page content.

  • Source data
    GitHub metadata grounds the copy.

    The real-data pass uses public GitHub metadata only: repo names, descriptions, languages, licenses, fork/source status, Pages, and update dates.

Readable when the source changes.

This page is a hand-curated snapshot, not a live dashboard. If GitHub is unavailable, the directory still gives visitors useful names, roles, and inspection links; if the org changes, the update path is the site repository.

Useful before polished.

The page makes one promise: Hermegeddon publishes work with source-first context, visible boundaries, and enough provenance to inspect later.
